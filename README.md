## SSH

```bash
git clone https://github.com/romanmikh/42_fractol.git fractol
cd fractol
make
./fractol
```

## Controls
- Zoom: Mouse scroll or m/n keys.
- Move: Arrow keys for panning.
- Change Fractals: Follow prompt instructions.
- Exit: Press ESC to quit.
